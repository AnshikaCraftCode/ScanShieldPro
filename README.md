
🛡️ ScanShield Pro

AI Threat Intelligence Engine

A lightweight, AI-powered scam detection web application built entirely with HTML, CSS, and JavaScript — delivering intelligent threat analysis with 99.2% accuracy.

📌 Overview

ScanShield Pro – AI Threat Intelligence Engine is a smart message scanning system that detects phishing, fraud, scam patterns, and malicious intent directly in the browser.

⚡ No backend

⚡ No server

⚡ No database

⚡ 100% Frontend-Based

Everything runs locally using JavaScript logic and pattern detection.

🎯 What It Detects

Phishing messages

Malicious URLs

IRS scam messages

Bank fraud alerts

Fake job offers

Prize / lottery scams

Suspicious keywords & patterns

Legit vs Scam classification

📊 What It Shows


After scanning, the system displays:

Scam Probability Percentage


Threat Category

Pattern Match Details

Recommended Security Actions

Total Threat Count

Total Scan Count

Model Accuracy: 99.2%

⚙️ How It Works


User enters a suspicious message.

JavaScript processes the input.

Pattern-matching engine checks for:


Scam keywords

Fraud indicators

Malicious links


Probability score is calculated.

The result is displayed with:

Risk level

Threat type

Recommended action

All logic runs directly in the browser.
